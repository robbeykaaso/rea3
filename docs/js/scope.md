# Abstract  
* an additional data container based on the stream  

# API  
* **QJSValue cache(const QString& aName, QJSValue aData)**  
    - cache the data on self  
    - `aName` is the data name  
    - `aData` is the data  
    - `return` self  
</br>

* **QJSValue data(const QString& aName)**  
    - `aName` is the data name  
    - `return` the data by name  
</br>
