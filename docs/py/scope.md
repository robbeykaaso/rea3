# Abstract  
* an additional data container based on the stream  

# API  
* **cache(self, aName: str, aData: any) -> 'scopeCache'**  
    - cache the data on self  
    - `aName` is the data name  
    - `aData` is the data  
    - `return` self  
</br>

* **data(self, aName: str) -> any**  
    - `aName` is the data name  
    - `return` the data by name  
</br>
