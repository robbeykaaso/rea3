# Abstract
* the search component for qml  

# Attribute
* text: the input text  
* hint: the placeholder text  
* prefix: the prefix text  
* signal accepted: it will be emitted when the mouse click or input is complete  
_sample_:  
```
    Search0{
        text: qsTr("hello")
        hint: qsTr("Enter...")
        prefix: "@"
    }
```
