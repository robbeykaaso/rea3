# Abstract
* the text edit component for qml  

# Attribute
* caption: Text object  
* background: Rectangle object: background  
* input: TextInput object: edit  
* ratio: double: the ratio between caption and background  
_sample_:  
```
    Edit{
        width: 180
        caption.text: qsTr("attr1") + ":"
        ratio: 0.4
    }
```
