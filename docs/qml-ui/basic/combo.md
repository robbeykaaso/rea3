# Abstract
* the combobox component for qml  

# Attribute
* caption: Text object  
* background: Rectangle object: background  
* combo: ComboBox object  
* ratio: double: the ratio between caption and background  
_sample_:  
```
    Combo{
        width: 180
        caption.text: qsTr("attr3") + ":"
        ratio: 0.4
    }
```
