# Abstract
the wrapper of gridder0  

# API Pipe
* **name + _updateViewCount**  
    - renew the grid count and layout  
    - input: array or number  
    - output: array or number  
    - type: pipe  
_sample_:  
```
Pipeline.run(gridder_cld.name + "_updateViewCount", {size: [5, 5]})
```
